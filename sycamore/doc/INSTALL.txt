Installing Sycamore
-------------------

Dependencies
============

Required
~~~~~~~~

 * Python >= 2.3 (http://python.org/)
 * PIL >= 1.1.5 (http://pythonware.com/products/pil/)
 * diffutils >= 2.8.1 (http://gnu.org/software/diffutils/)
 
 * MySQL: 
   * MySQL >= 5.0 (http://mysql.com/) 
   * MySQLdb >= 1.2.1c3 ()

 * or PostgreSQL:
   * PostgreSQL >= 7.4 (http://postgresql.org/)
   * psycopg2 >= 2.0 (http://initd.org/projects/psycopg2)

Optional
~~~~~~~~

 * xapian-core >= 0.9.4 (http://xapian.org)
 * xapian-bindings >= 0.9.4 (http://xapian.org)
